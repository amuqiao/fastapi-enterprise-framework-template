# FastAPI企业级框架模板 - 示例集合

本目录包含了FastAPI企业级框架模板的各种使用示例，帮助你快速上手并了解框架的核心功能。

## 示例列表

### 基础示例
- **[快速入门](./quickstart/)**：创建一个简单的API应用，了解框架的基本使用方法
- **[配置管理](./config/)**：学习如何使用框架的配置管理功能
- **[中间件使用](./middleware/)**：了解如何使用和自定义中间件

### 核心功能示例
- **[认证系统](./auth/)**：实现完整的用户认证系统，包括JWT认证、OAuth2密码流等
- **[数据库操作](./database/)**：使用数据库抽象层进行CRUD操作，实现仓储模式
- **[事件系统](./events/)**：学习如何使用框架的事件总线，实现事件驱动架构

### 高级示例
- **[测试策略](./testing/)**：了解如何为FastAPI应用编写测试
- **[部署指南](./deployment/)**：学习如何部署FastAPI应用到生产环境

## 如何使用示例

1. **克隆仓库**：确保你已经克隆了FastAPI企业级框架模板仓库
2. **进入示例目录**：`cd examples/<示例名称>`
3. **安装依赖**：`pip install -r requirements.txt`
4. **运行应用**：`python main.py`
5. **访问API文档**：打开浏览器访问 `http://localhost:8000/docs`
